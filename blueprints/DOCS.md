# DOCS.md: Documentation Roadmap

#### **Documentation Structure**
*   **Getting Started:** Setup instructions for local development and API keys.
*   **Scraper Handbook:** Documentation on how to update `ProspectParkScraper` and `NYRRCollector` when site layouts change.
*   **Impact Engine Logic:** Detailed math and weighting behind the Impact Scores.
*   **API Reference:** Full documentation for `/v1/alerts` and `/v1/events` (see `API.md`).

#### **Maintenance**
Documentation is kept in the `docs/` directory and is automatically rebuilt or updated alongside code changes to ensure it remains the source of truth.
